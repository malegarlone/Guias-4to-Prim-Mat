Reglas de Oro

Para escribir números romanos, seguimos dos reglas principales:

1. Suma: Si una letra menor va después de una mayor, se suma.
VI = 5 + 1 = 6
2. Resta: Si una letra menor va antes de una mayor, se resta.
IV = 5 - 1 = 4

Recuerda: ¡No puedes repetir una letra más de 3 veces seguidas!

Niveles de Sabiduría

Los mayas escriben de abajo hacia arriba. Cada nivel multiplica el valor:

4º Nivel
(x 8,000)
3º Nivel
(x 400)
2º Nivel
(x 20)
1º Nivel
(x 1)

En este ejemplo: (1 x 20) + (5 x 1) = 25
